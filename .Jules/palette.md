## 2026-05-04 - [Accessibility and Clarity for Technical Acronyms]
**Learning:** In technical applications (like AAMVA barcode generators), acronyms (DCG, DAC, etc.) are common but opaque to non-expert users. Using Streamlit's `help` parameter provides an unobtrusive way to offer definitions without cluttering the UI.
**Action:** Always provide descriptive tooltips for technical field acronyms using the `help` parameter in Streamlit components to ensure the interface is accessible and self-documenting.
