## 2025-05-10 - [Technical Acronym Tooltips]
**Learning:** In technical domain-specific interfaces (like AAMVA standards), using only acronyms as labels creates high cognitive load for non-expert users. Streamlit's `help` parameter is an effective way to provide immediate context without cluttering the UI.
**Action:** Always check for technical shorthand or acronyms in input labels and provide descriptive tooltips via the `help` parameter to improve accessibility and user confidence.
